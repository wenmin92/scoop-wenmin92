Question
--------

**1. How to install the apps from this bucket?**

Run below command in PowerShell to add the bucket:

``` powershell
scoop bucket add wenmin92 https://github.com/wenmin92/scoop-wenmin92
```

Install apps from this bucket with below command:

``` powershell
scoop install wenmin92/<app_name>
```

**2. I want some other apps!**

Please open new app request [issue](https://github.com/wenmin92/scoop-wenmin92/issues). :)

**3. Some apps are outdated, please update it!**

Be a contributor, man! Fork it, update the outdated apps app manifest, and file pull-request. :D
